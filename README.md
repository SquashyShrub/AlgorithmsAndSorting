# Common Sorting & Searching Algorithms
**Current Version:** `v0.0.1-stable` | **Last Updated:** 2026-4-23
> A sorting and searching algorithm visualizer built for ASU SER 222.

---

### Overview
Developed during my **2nd Year** at Arizona State University. This project focuses on **practicing common sorting and searching patterns used commonly in interviews and DSA exams**. 

### 💻 Tech Stack
- **Language**: C# & Java
- **Frameworks/Tools**: .NET, Java SDK
- **Environment**: Visual Studio Code

### 🛠️ Installation & Usage
1. Clone the repo: `git clone [url]`
2. Open in VS Code or Visual Studio.
3. Create a class of the algorithm you want to test.
4. Press `F5` or `Run` to execute.
